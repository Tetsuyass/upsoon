Display a custom status banner across your site to communicate maintenance, updates, or downtime to your visitors.

== Description ==

**UpSoon** is a lightweight and customizable plugin that allows you to display a small banner at the top or bottom of your WordPress site.

You can use it to:
- Notify users about site maintenance or downtime
- Communicate important updates (e.g., new features, new policy, temporary issues)
- Customize the style and position to fit your site’s design

**Ideal for small projects or personal blogs**, the plugin is easy to use and does not require any third-party dependencies.

== Features ==

- Show a customizable banner at the top or bottom of your site
- Change the message, background color, text color
- Enable/disable the banner with one click
- Set visibility for visitors or logged-in users
- Responsive and accessible design

== Installation ==

1. Upload the plugin folder `UpSoon` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Go to "Settings" > "Status Banner" to configure your message and options.

== Frequently Asked Questions ==

= Can I display the banner only for logged-in users? =
Yes! You can configure this option in the plugin settings panel.

= Will it slow down my site? =
Not at all. The plugin is minimal and optimized for performance.
